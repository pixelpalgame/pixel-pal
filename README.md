# Pixel Pal

A pixel-art life-sim RPG set in a living city. Explore the world, gather resources, craft gear, fight enemies, and take on bosses — solo or with a friend online.

---

## Install & Play

1. Double-click **INSTALL.bat** — sets everything up automatically (one time only)
2. A **Pixel Pal** shortcut will appear on your Desktop
3. Double-click it any time to play

## Keeping the Game Updated

Run **UPDATE.bat** to check for and install the latest version. Your save data is never affected.

---

## Features

- **Open world** — explore the city, docks, park, mines, and downtown strip
- **Crafting** — forge weapons, shields, tools, and gear at the crafting bench
- **Mining** — gather ore, wood, fiber, and stone from Stonepick Mines
- **Combat** — fight enemies on the Downtown Strip with a full attack, block, and dodge system
- **Boss fight** — defeat the Downtown Warden to complete the demo
- **Online co-op** — play with a friend on the same network via the in-game lobby
- **AI companion** — your character reacts to the world and converses using a built-in AI
- **World map** — fast travel between all locations
- **XP & skills** — level up combat, mining, gathering, and more
- **Save system** — auto-saves progress with manual save and backup options

## Controls

| Key | Action |
|-----|--------|
| `← → / A D` | Move |
| `Space / W` | Jump |
| `E` | Interact / Attack / Mine |
| `Q` | Block |
| `M` | World map |
| `TAB` | Inventory |
| `G` | Equipment |
| `C` | Crafting |
| `F11` | Toggle fullscreen |
| `ESC` | Leave scene / close panel |

## Online Co-op

Both players need to have the game installed and running on the same network.

1. **Host:** click **PLAY ONLINE** → **HOST GAME** → **CREATE ROOM** → share your IP and room code
2. **Guest:** click **PLAY ONLINE** → **JOIN GAME** → enter the host's IP and room code → **JOIN ROOM**
